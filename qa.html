客戶資金在自己交易所帳戶，TRL不代為保管，可隨時決定進場或出場。API Key 設定上僅開啟「讀取」與「現貨交易」權限（不開啟提幣權限），並搭配交易對白名單（防止被盜用去對敲冷門小幣對）與 IP 白名單雙重防護，進一步確保帳戶與資金安全無虞。

客戶需自己使用固定IP，如果使用浮動IP必需每隔3-5天把新的浮動IP增加到交易所的API白名單，以確保機器人能持續連線下單。

客戶電腦需24小時開機，機器人才會持續運作交易。在自己電腦上跑不需要另外購買雲端空間，省了雲端費同時也增加安全。

目前只接受 USDT 或 USDC 付款。

支援 Binance、OKX、Bybit、Bitget；其中 OKX 和 Bitget 除了 API Key/Secret 外，還需要額外輸入 Passphrase。

機器人會自動安全停止（不會強制賣出持倉、不會影響現有部位），只是暫停監控與下新單，重新輸入有效金鑰即可接續運作。

不會自動開機，需要人工重新啟動電腦並開啟程式；建議設定開機自動啟動，若擔心斷電建議搭配不斷電系統（UPS）。

機器人僅依照個人策略設置執行，不保證獲利。交易涉及風險，過往績效不代表未來表現，本服務不構成投資建議。

取決於投入資金與市場波動以及個人策略設置都會影響，不能保證多久回本。

請聯繫客服並提供付款時的轉帳紀錄（TxID），協助核對身分後可重新核發金鑰。
